== Description ==

Processe pagamentos conosco. PIX, boleto, e cartões de crédito. Confira agora a WEpayments. Seu parceiro para pagar, receber e gerenciar pagamentos online no Brasil.


= Instalação =

Confira o nosso guia de instalação e configuração na aba [Installation](http://wordpress.org/extend/plugins/wepayments/installation/).

= Compatibilidade =

- Requer WooCommerce 6.0 ou posterior para funcionar;
- Requer Brazilian Market on WooCommerce para funcionar;

= Ficou com alguma dúvida? =

Você pode:

- Acessar nosso FAQ logo abaixo;
- Criar um tópico no [fórum de ajuda do WordPress](https://wordpress.org/support/plugin/wepayments/);
- Entrar em contato com a gente pela nossa [Central de Ajuda](https://wepayments.com.br/contato).

== Installation ==

Para instalar o nosso plugin são apenas 2 passos simples:

- Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.
- Ative o plugin.

= Como fazer a ativação =

#### **No WooCommerce**

- Configura o plugin na aba WooCommerce > Configurações > Integração > WEpayments;
- Insira suas chaves;
- Na WEpayments, temos o ambiente para testes, chamado Sandbox, e o ambiente de Produção, que é o definitivo. Se você estiver com os tokens de produção, precisa desmarcar a caixinha de sandbox no WordPress para começar a vender!

= Requerimentos =

Para garantir a compatibilidade do seu e-commerce com o plugin da WEpayments, você precisa dos seguintes requerimentos:

- Requer WooCommerce 6.0 ou posterior para funcionar;
- Requer Brazilian Market on WooCommerce para funcionar;
- PHP 7.4 ou superior.

= Configurações do plugin =

Para configurar as chaves de integração da WEpayments, acesse a aba do WooCommerce > Configurações > Integração > WEpayments

== Frequently Asked Questions ==

= Qual é a licença do plugin? =

Este plugin esta licenciado como GPL.

= O que eu preciso para utilizar este plugin? =

* WooCommerce 7.0 ou posterior.
* Brazilian Market on Woocommerce.

= Consigo usar o ambiente sandbox =

Sim, nas configurações de integração você poderá selecionar o sandbox em "WooCommerce" > "Configurações" > "Integração" > "WEpayments"

= Consigo habilitar somente o cartão de crédito, pix ou boleto? =

Sim, você poderá habilitar os dois meios de pagamento ou apenas um deles, na configuração padrão do WooCommerce em "WooCommerce" > "Configurações" > "Pagamentos"
